# Protocols

## Key Protocols to Track

### AMM Protocols
- **Uniswap**: Pioneer of AMMs and concentrated liquidity
- **Curve**: Specialized for stablecoin and correlated asset swaps
- **Balancer**: Multi-asset pools and customizable weights
- **SushiSwap**: Community-driven AMM with additional incentives
- **PancakeSwap**: BSC-focused AMM with concentrated liquidity

### Lending Protocols
- **Aave**: Pool-based lending with flash loans and governance
- **Compound**: Utilization-based interest rate models
- **MakerDAO**: CDP-based stablecoin lending and liquidation
- **Morpho**: Peer-to-peer matching on top of pool protocols
- **Euler**: Isolated lending markets and risk management innovation

### MEV / Infrastructure
- **Flashbots**: Dominant MEV infrastructure and private relay network
- **BloXroute**: Private transaction relays and MEV tools
- **MEV-Boost**: Builder/relay separation for Ethereum validators
- **CowSwap**: Batch auctions and MEV-resistant order flow

### Oracle Protocols
- **Chainlink**: Most widely used decentralized oracle network
- **Pyth**: High-frequency price feeds, especially for derivatives
- **Tellor**: Dispute-based decentralized oracle mechanism
- **Band Protocol**: Cross-chain oracle data provider

## Notes

- Monitor protocol parameter changes over time
- Compare liquidation thresholds and rate model changes
- Watch for governance proposals affecting economic design
- Track security incidents and oracle failures

## Suggested Tracking Framework

For each protocol, record:
- Core mechanism
- Capital efficiency model
- Oracle design
- Liquidation process
- Fee and incentive structure
- Known exploits / incidents
